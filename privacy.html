Privacy Policy & Terms (Demo)

This is a demonstration prototype. No real user data is transmitted to external servers by this demo.

Data Collected (Demo)
  • Local session/demo storage for tourist information and alerts.
  • Local geolocation (with user permission) when Panic is pressed; it is not sent to any server in this demo.
Security

Production systems should use end-to-end encryption, secure authentication, and comply with applicable data protection laws (e.g., India's IT/Privacy regulations).

Consent

By using the live system (production), users should be shown a detailed consent dialog explaining what is collected, retention periods, and opt-out options.
